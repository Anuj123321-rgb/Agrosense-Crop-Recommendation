# 🌾 AgroSense - Crop Recommendation System

## 🚀 Overview

AgroSense is a machine learning-based web application that recommends the best crop to grow based on soil and environmental conditions.

## 🧠 Features

* ML-based crop prediction (99% accuracy)
* Ensemble model (Extra Trees + XGBoost + LightGBM)
* Real-time prediction via Flask API
* Feedback system for continuous learning
* Auto-retraining pipeline
* Interactive UI

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-learn
* XGBoost, LightGBM
* SQLite
* HTML/CSS/JS

## ⚙️ How to Run Locally

```bash
pip install -r requirements.txt
python database.py
python train_model.py
python app.py
```

## 🌍 Deployment

Deployed using Render (Free Cloud Platform)

## 📊 Model Performance

* Accuracy: ~99%
* Dataset: Crop Recommendation Dataset (2200+ samples)
